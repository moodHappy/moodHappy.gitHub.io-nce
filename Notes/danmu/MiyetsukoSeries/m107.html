美悦子系列

第一阶段：《白夜双姝》

核心：极致纯白与深渊顺从
状态：天鹅vs容器

关键人物

  • 美悦子：林家最后的天鹅公主；被剥夺姓名的编号107，审计报表中核销的残次耗材，反噬债主的呆滞坏账；极乐坂的肖邦，妖艳女皇，影子帝国的精算师，九条岚子的掘墓人；桐生家的绞刑吏；白凤家的宿命债主；御影沙织最后的持有者，布谷与花乃井的意志握柄；黑崎宗一郎的法律正妻，里约的白皇后，合欢殿的纵火者；全球资产的终极清算人；死亡法则的守护者，以及——陈默唯一的、不带编号的爱人。
  • 美月眠： 意志真空的溺水女神，她是林家帝国最致命的溃口。​作为林家的首席秘书，她将绝对顺从化为生理本能，在掠夺者的占有中，如容器般无意识地溢出核心机密与资产。她是美悦子命运的镜像：一个因丧失拒绝权而彻底物化、被欲望贯穿至透明的温热空壳。
    沙织的美的本质：如水般的放弃。当一个绝色女人彻底放弃了自我的意志，她就成了一种超越道德的、纯粹的生理奇观。她是那种能让黑崎、白凤这种阅尽千帆的掠夺者都感到沉醉其中，也曾让东京财阀之首藤堂英司死在她床上。桐生岳更是评：纵有千座合欢殿，不如一个美月眠。
  • 陆野拓海 (变数)： 带着薄荷味的转校生，运动万能。他是打破美悦子玻璃罩的“破窗者”，代表了真实、廉价却温暖的世界。
  • 小健 (情报员)： 拓海的好友，技术宅。他通过摄像头和监听无意中触碰到了 B 面的肮脏真相。
  • 绘里子 (嫉妒者)： 校园社交女王，负责在明面上给美悦子制造各种充满少女感的危机。
  • 黑崎宗一郎(终极买家)： 垂死的旧帝国掠夺者。一个阳痿、暴戾、却掌控着绝对金钱权力的老财阀。他是“107号评估体系”的终极买家。他看中沙织的“顺从”，更渴望摧毁美悦子的“傲骨”。他亲手签发的贬值报告，是将林家资产彻底物化、数据化的法律宣判。

世界观设定

  • 奴妻： 不对外公开的秘密妻子。“编号”仅在黑崎品尝初夜时使用，作为彻底剥夺人格、将其物化的符号。
  • 容器： 美月眠生理性的绝对顺从体。她没有拒绝的机能，是任何人皆可进入的“公共接口”。她在被占有的快感与痛楚中无意识地溢出林家机密；只要向她索取，她都会带着那种悲剧性的微笑，提供对方渴望的一切。
  • 极乐坂： 权贵口中的合欢殿，底层嘲讽的苟合殿。它是帝国金融博弈的血色暗面，一座焚烧名媛尊严与核心机密的焚尸炉；在这里，顶级资产评估与肉体损耗同步进行，是文明在欲望面前彻底崩溃后的终极祭坛。
  • 残月楼：一个被编织进都市传说、游离于可观测世界之外的禁忌维度。它的存在本身就是一个悖论：人们在密谈中敬畏它，却无人能确证其坐标。传闻中此地不闻欢声，亦无呻吟，唯有一抹苍白残月常悬其上，俯瞰着关于国运、命脉与因果的寂静博弈。入者无生，唯余死寂。

第二阶段：《肉债》

核心： 蜘蛛之网
状态： 编号“107”，极乐坂的女皇，极乐坂的会长，黑崎夫人，藏在黑暗里的操盘手。

关键人物

  • 花乃井 (编号 209)： 极乐坂的新人，由美悦子亲手调教，从只有在被黑崎“品尝”时才被称为 209，平日里她是美悦子最信任的执行工具。
  • 布谷 (Cuckoo)： 潜伏于白凤集团核心的内线。他曾是林家旧部的后代或受过林家恩惠，对黑崎与白家有着深层仇恨。
  • 九条岚子： 极乐坂前会长，后美悦子取代。
  • 桐生岳： 黑崎岳父，后破产自杀。
  • 桐生纱夜： 黑崎原配，桐生岳之女，因其父失势后被丢弃。
  • 白凤雅人： 黑崎最强大的盟友，白凤雅人政商丑闻中流亡海外。
  • 秘密共谋： 美悦子在极乐坂内部促成了布谷与花乃井的联手。两人作为美悦子的影之刃，以“自由”为代价，甘愿在白家内部埋下火种。
  • 幕后操盘： 美悦子一边通过服侍匿名客户标记“死亡名单”，一边通过布谷与花乃井实施具体的实地渗透，导演黑崎与白家的内斗。

    目标：不可用 (S1 - S2：皮肉生涯)

    执行： 美悦子、布谷、花乃井

    • 事件 1：【编号仪式】 黑崎宗一郎在那份评估报告上画下红叉。被迫换上印有“107”的廉价制服，这标志着“林美悦子”的社会性死亡。
    • 事件 2：【极乐坂初夜审计】 面对第一个粗鄙的顾客，美悦子几乎崩溃。花乃井在隔壁房间用木板的敲击声引导她完成“人格剥离”，教会她如何像机器一样工作。
    • 事件 3：【布谷登场】 影之刃就位。
    • 事件 4：【花乃井(编号 209)登场】 调教妓艺。
    • 事件 5：【废墟溢价】 市场上流传着“末代千金”的传闻。美悦子利用这种扭曲心理，在花乃井操盘下维持“帝国废墟”的破碎感，迅速积累第一笔属于自己的暗财。
    • 事件 6：【数据窃取】 美悦子利用接客间隙窃取政商显要随身文件碎片，由花乃井整合，建立极乐坂第一份“地下情报网”。
    • 事件 7：【肖邦的夜曲】 在一个弥漫着昂贵雪茄与廉价情欲的私密沙龙里，一位曾仰慕林家权势的政客指名要107号弹奏钢琴，试图在旋律中彻底羞辱这位“坠落的天鹅”。此夜过后，“极乐坂的肖邦”之名在顶级权贵圈疯传，美悦子从一个平庸的“残次品”瞬间爆火为“孤品级耗材”。

    阶段结局： 107号从残次品变成极乐坂的头牌。在肉体磨损之中，美悦子的灵魂完成第一次冷酷结晶。

    目标：九条岚子 (S3 - S5：旧神陨落)

    执行： 美悦子、花乃井、布谷

    • 事件 1：【致命审计】 美悦子在黑崎面前“无意”发现岚子的财务漏洞，诱导黑崎产生怀疑，引发内斗。
    • 事件 2：【血色姐妹会】 岚子试图处决花乃井以儆效尤。美悦子联合所有被压榨的编号女孩，发动集体沉默抗议。
    • 事件 3：【账本掉包】 述职之夜潜入密室，替换真实账本为伪造贪腐证据，切断岚子的政治退路。
    • 事件 4：【权力移交】 岚子众叛亲离被废黜。美悦子以“最稳健创收者”身份戴上黑色烟杆，成为新会长。

    阶段结局： 九条岚子消失于东京湾迷雾。美悦子掌握全部现金流，黑化为真正的“极乐坂女皇”。

    目标：桐生岳、桐生纱夜 (S6 - S8：围魏救赵)

    执行： 布谷、花乃井

    • 事件 1：【影子绞杀】 布谷潜入桐生帝国核心财务部门，配合外部做空，引爆短期债权雷区。
    • 事件 2：【公主的丑闻】 获取纱夜秘密私会证据，在黑崎与桐生合作关键节点发布，使其瞬间贬值。
    • 事件 3：【联姻要挟】 桐生岳走投无路。美悦子以白衣骑士姿态现身，用极乐坂三年现金流交换那张正妻结婚证。
    • 事件 4：【身份回溯】 大婚前夜，黑崎在惊悚暗示中，隐约记起多年前名为“107”的残次品。

    阶段结局： 桐生岳破产自杀，纱夜被丢弃。美悦子加冕为“黑崎夫人”，完成从奴妻到正妻的血色复位。

    目标：白凤雅人 (S8 - S10：焚城之火)

    执行： 布谷、花乃井

    • 事件 1：【白凤的陷阱】 白凤察觉真实意图，试图以“容器”沙织为人质逼迫交权。
    • 事件 2：【极乐坂暴动】 美悦子下令焚毁全部档案，一次性引爆数十年政商丑闻，城市权力结构陷入火海。
    • 事件 3：【最后的猎杀】 布谷执行斩首行动，与白凤私人武装同归于尽。
    • 事件 4：【幻灭的守护】 花乃井为销毁最后罪证，带着秘密消失于废墟。

    阶段结局： 白凤势力灰飞烟灭。美悦子虽胜，却失去最后温情。进入伊豆疗养院，在海浪声中遇见生命最后的变数——陈默。

    逻辑点评：
    • 关于花乃井： 她是美悦子的“外脑”，负责经济与信息层面的黑暗博弈。
    • 关于布谷： 他是美悦子的“利刃”，当逻辑失效时，以血与暴力开路。
    • 关于精神重创： 美悦子的胜利本质是不断“自我截肢”。她赢得世界，却成为孤家寡人。这种极致寂寥，完美衔接伊豆疗养院篇章。

第三阶段：《女王的法则》

核心： 涅槃归来与全球清算
状态： 掌控秩序的女王，与共生伙伴陈默的最终一击。

契约开启：“偷金库”事件

  • 行动： 伊豆相识后，美悦子与陈默联手劫走黑崎家的地下秘密金库。这不仅是资本的原始积累，更是两人亲密伙伴关系的确立。

崛起与反攻

  • 东京大逃亡： 劫案后，两人在全城搜捕中杀出血路，彻底人间蒸发。
  • 里约嘉年华： 两人空降里约，在狂欢节的混乱中完成原始积累与权力重组，美悦子正式淬炼为“女王”。

终极审判

  • 黑崎的覆灭： 重返东京，正面摧毁一切恶的源头——黑崎宗一郎（本系列最大 Boss）。
  • 三岛的倒台： 扫清黑崎余部及新兴对手三岛，确立唯一的女王秩序。
  • 死亡法则： 全球清算。根据《肉债》时期的名单，对当年的匿名客户进行定点物理猎杀。

流浪篇 (The Wandering)

规划中： 尘埃落定后，美悦子与陈默在世界边缘的自我流浪与最终救赎。
