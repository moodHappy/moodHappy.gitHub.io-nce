明朝君主列表

大明王朝

大明（1368–1644） 天子守国门君王死社稷，大明风华十六帝。
  • 朱元璋 （洪武帝 / 太祖，开局一个碗，驱逐胡虏恢复中华）
  • 朱允炆 （建文帝，削藩引发靖难之役，下落不明）
  • 朱棣 （永乐帝 / 成祖，靖难篡位，迁都北京，郑和下西洋）
  • 朱高炽 （洪熙帝 / 仁宗，在位仅十个月，开创仁宣之治）
  • 朱瞻基 （宣德帝 / 宣宗，促成仁宣之治，人称“促织天子”）
  • 朱祁镇 （正统帝 / 英宗，土木堡之变被俘，叫门天子）
  • 朱祁钰 （景泰帝 / 代宗，临危受命，北京保卫战）
  • 朱祁镇 （天顺帝 / 英宗，夺门之变复辟）
  • 朱见深 （成化帝 / 宪宗，专宠万贵妃，设立西厂）
  • 朱祐樘 （弘治帝 / 孝宗，弘治中兴，中国历史上罕见的一夫一妻制皇帝）
  • 朱厚照 （正德帝 / 武宗，极具争议的顽童皇帝，自封“威武大将军”）
  • 朱厚熜 （嘉靖帝 / 世宗，修仙炼丹，几十年不上朝，引发壬寅宫变）
  • 朱载坖 （隆庆帝 / 穆宗，隆庆开关，俺答封贡）
  • 朱翊钧 （万历帝 / 神宗，万历三大征，后期近三十年不上朝）
  • 朱常洛 （泰昌帝 / 光宗，在位仅一月，红丸案暴毙）
  • 朱由校 （天启帝 / 熹宗，木匠皇帝，魏忠贤专权）
  • 朱由检 （崇祯帝 / 思宗，勤政亡国，李自成破京师，自缢于煤山）

南明（抗清政权）

南明（1644–1662） 仓皇南渡欲求偏安，内斗不休山河飘絮。
  • 朱由崧 （弘光帝，建都南京，史可法扬州殉国，亡于清）
  • 朱聿键 （隆武帝，建都福州，南明较有作为的君主，绝食而死）
  • 朱聿鐭 （绍武帝，建都广州，与永历政权内斗，自缢殉国）
  • 朱由榔 （永历帝，建都肇庆/桂林等，李定国抗清，后逃亡缅甸被吴三桂绞杀）
