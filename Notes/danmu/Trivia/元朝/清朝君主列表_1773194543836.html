清朝君主列表

一、后金（崛起关外）

后金（1616–1636） 十三遗甲白山黑水，萨尔浒战奠定辽东。
  • 努尔哈赤 （天命汗 / 太祖，统一女真各部，建立后金）

二、大清（定鼎中原）

大清（1636–1912） 康乾盛世耀东亚，落日余晖丧权辱。
  • 皇太极 （崇德帝 / 太宗，改女真为满洲，改国号大清）
  • 福临 （顺治帝 / 世祖，清军入关，定鼎北京）
  • 玄烨 （康熙帝 / 圣祖，擒鳌拜平三藩，收复台湾，康乾盛世开端）
  • 胤禛 （雍正帝 / 世宗，承上启下，设立军机处，推行摊丁入亩）
  • 弘历 （乾隆帝 / 高宗，十全武功，版图极盛，但后期闭关锁国走向衰落）
  • 颙琰 （嘉庆帝 / 仁宗，扳倒和珅，镇压白莲教起义，清朝由盛转衰）
  • 旻宁 （道光帝 / 宣宗，签《南京条约》，鸦片战争爆发，中国近代史开端）
  • 奕詝 （咸丰帝 / 文宗，遭遇太平天国运动与英法联军火烧圆明园）
  • 载淳 （同治帝 / 穆宗，慈安、慈禧两宫太后垂帘听政，洋务运动同治中兴）
  • 载湉 （光绪帝 / 德宗，甲午海战惨败，戊戌变法失败后被囚禁瀛台）
  • 溥仪 （宣统帝 / 逊帝，三岁登基，辛亥革命后退位，两千余年封建帝制终结）
