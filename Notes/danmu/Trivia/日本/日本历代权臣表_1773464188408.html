日本历代权臣表

飞鸟 - 奈良时代 (氏族强权与外戚萌芽)
权臣 / 掌权家族 傀儡/共治天皇 核心事迹与掌权手段
苏我马子
大臣		 第32代 崇峻天皇
第33代 推古天皇		 日本古代头号权臣。因权力斗争直接派人暗杀了崇峻天皇，随后拥立自己的外甥女推古天皇，大权独揽。
苏我入鹿
大臣		 第35代 皇极天皇 苏我氏专横的巅峰，威势甚至盖过天皇。最终在“乙巳之变”中被中大兄皇子当场斩首。
藤原不比等 第43代 元明天皇 藤原氏繁荣的奠基人，制定《大宝律令》，将女儿嫁给天皇，开启了藤原氏靠“送女儿进宫当外戚”控制朝政的剧本。
道镜
法王 (妖僧)		 第48代 称德天皇 靠医疗和宗教手段极度控制女天皇，被封为“法王”，差一点通过神托伪造篡夺皇位，是日本历史上最接近改朝换代的外人。
平安时代 (摄关政治、院政大佬与平氏崛起)
权臣 / 掌权家族 傀儡/共治天皇 核心事迹与掌权手段
藤原良房
摄政		 第56代 清和天皇 日本历史上第一位人臣摄政。趁天皇年幼，以外祖父身份把持朝政，正式拉开摄关政治序幕。
藤原基经
关白		 第57代 阳成天皇 日本历史上第一位关白。嫌弃阳成天皇性格暴戾不听话，直接以天皇精神不正常为由将其强行废黜，另立新君。
藤原道长
摄政 / 关白		 第66代 一条天皇等 藤原氏专权的绝对巅峰。搞出“一家立三后”，留下名言“此世即吾世，如月满无缺”，天皇沦为藤原家的生育工具。
白河 / 后白河法皇
治天之君 (院政)		 第73代 堀河天皇等
多位幼年天皇		 虽是皇室，但首创“院政”，退位出家为法皇并掌握实权，把在位天皇当傀儡。后白河法皇更是挑动源平合战，被称为“日本第一大天狗”。
平清盛
太政大臣		 第80代 高仓天皇
第81代 安德天皇		 首个开创武家政权的武士。打破贵族垄断，软禁后白河法皇，将外孙安德天皇推上皇位。曾言：“非平氏者非人也”。
镰仓时代 (源氏断绝与北条氏窃国)
权臣 / 掌权家族 傀儡/共治天皇 核心事迹与掌权手段
源赖朝
征夷大将军		 第77代 后白河天皇
第82代 后鸟羽天皇		 消灭平氏，建立镰仓幕府，逼迫朝廷交出全国土地管理权和警察权。从此日本进入长达近700年的武士统治时代。
北条政子
尼姑将军		 第82代 后鸟羽天皇
(对抗朝廷)		 源赖朝正室。源氏绝嗣后，她掌控幕府实权。在承久之乱前发表震撼演讲凝聚天下武士，大败天皇官军，稳固了北条氏的窃国基业。
北条义时
第2代执权		 第82代 后鸟羽天皇
第85代 仲恭天皇		 面对后鸟羽上皇发动的倒幕战争，率领武士大败官军。直接流放了三位太上皇，废黜了在位仅78天的仲恭天皇，朝廷彻底沦为幕府下属。
北条时宗
第8代执权		 第90代 龟山天皇
第91代 后宇多天皇		 面临元朝大军的两次入侵（文永·弘安之役），以强硬铁腕斩杀元使，倾全国之力抗元，并在“神风”帮助下保全了日本。
室町 - 战国时代 (武家大分裂与下克上狂潮)
权臣 / 掌权家族 傀儡天皇 (及将军) 核心事迹与掌权手段
足利尊氏
征夷大将军		 第96代 后醍醐天皇
北朝历代天皇		 背叛镰仓幕府后又背叛后醍醐天皇。把自己赶到吉野山（南朝），在京都拥立北朝天皇，建立室町幕府，开启南北朝时代。
足利义满
太政大臣 / 大将军		 第100代 后小松天皇 逼迫南朝交出神器统一日本。接受明朝“日本国王”封号。晚年甚至想架空天皇让自己的儿子篡位，后暴毙而亡。
细川政元
幕府管领 / 半将军		 第104代 后土御门天皇
(及其幕府将军)		 发动“明应之变”，带兵废黜第10代幕府将军足利义材，另立新将军。开启了臣子随意废立将军的先河，拉开战国“以下克上”的序幕。
三好长庆 / 松永久秀
畿内霸主		 第105代 后奈良天皇
第106代 正亲町天皇		 织田信长之前的“天下人”。将足利将军赶出京都，其家臣松永久秀更是派兵攻打二条城，逼死第13代“剑豪将军”足利义辉。
安土桃山 - 江户时代 (天下人、幽禁皇权与幕末倒阁)
权臣 / 掌权家族 傀儡/共治天皇 核心事迹与掌权手段
织田信长
右大臣		 第106代 正亲町天皇 战国魔王。虽然靠给朝廷捐钱恢复了皇室生计，但完全将天皇视为统治工具，甚至曾计划强迫正亲町天皇退位。后死于本能寺之变。
丰臣秀吉
关白 / 太阁		 第107代 后阳成天皇 以平民之身就任摄关家的最高职位“关白”。挟天子以令诸侯，将天皇变成自己统一全日本的金字招牌。
德川家康 / 德川秀忠
江户幕府将军		 第108代 后水尾天皇 颁布《禁中并公家诸法度》，用法律明文规定天皇唯一的任务就是“做学问”，剥夺一切政治权力。干预“紫衣事件”，逼得天皇赌气让位。
井伊直弼
幕府大老		 第121代 孝明天天皇 面对黑船来航，无视天皇敕令独断签订不平等条约。发动“安政大狱”大肆屠杀尊王攘夷派，最终在江户城门外被暗杀（樱田门外之变）。
萨长同盟
维新元老 (倒幕派)		 第122代 明治天皇 打着“尊王攘夷”旗号推翻幕府，挟持年幼的明治天皇作为政治筹码。发布《王政复古大号令》，彻底终结武士政权，自身成为新时代的掌权巨头。
