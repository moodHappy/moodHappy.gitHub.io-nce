五代十国君主列表

注：标为 红色 的为该时期疆域较广、国力相对强大的政权。

一、五代 (907–960)

1. 后梁 (907–923) 篡唐建业开乱局，父子相残国祚倾。
  1. 朱温 — 太祖
  2. 朱友珪 — （废帝）
  3. 朱友贞 — 末帝
2. 后唐 (923–936) 百战灭梁振唐风，伶人乱政一场空。
  1. 李存勖 — 庄宗
  2. 李嗣源 — 明宗
  3. 李从厚 — 闵帝
  4. 李从珂 — 末帝
3. 后晋 (936–947) 割地称儿遗万臭，契丹铁骑毁山河。
  1. 石敬瑭 — 高祖
  2. 石重贵 — 出帝
4. 后汉 (947–950) 立国最短仅四载，猜忌杀将速覆亡。
  1. 刘知远 — 高祖
  2. 刘承祐 — 隐帝
5. 后周 (951–960) 励精图治显英主，陈桥兵变作嫁衣。
  1. 郭威 — 太祖
  2. 柴荣 — 世宗
  3. 柴宗训 — 恭帝

二、十国 (897–979)

1. 吴国 (902–937) 淮南霸业奠基石，权臣当道禅南唐。
  1. 杨行密 — 太祖
  2. 杨渥 — 烈宗
  3. 杨隆演 — 高祖
  4. 杨溥 — 睿帝
2. 南唐 (937–975) 文采风流冠江南，错失良机作楚囚。
  1. 李昪 — 烈祖
  2. 李璟 — 元宗
  3. 李煜 — 后主
3. 吴越 (907–978) 保境安民筑海塘，顺应天命纳土宋。
  1. 钱镠 — 太祖
  2. 钱元瓘 — 世宗
  3. 钱弘佐 — 成宗
  4. 钱弘倧 — 忠逊王
  5. 钱弘俶 — 忠懿王
4. 楚国 (907–951) 商贾立国聚金银，众驹争槽尽散尽。
  1. 马殷 — 武穆王
  2. 马希声 — 衡阳王
  3. 马希范 — 文昭王
  4. 马希广 — 废王
  5. 马希萼 — 恭孝王
  6. 马希崇 — （末代国君）
5. 闽国 (909–945) 开闽繁华成旧梦，骨肉相残海邦倾。
  1. 王审知 — 太祖
  2. 王延翰 — 嗣王
  3. 王延钧 — 惠宗
  4. 王继鹏 — 康宗
  5. 王延羲 — 景宗
  6. 王延政 — 天德帝
6. 南汉 (917–971) 酷刑峻法惧群臣，阉党治国千古笑。
  1. 刘䶮 — 高祖
  2. 刘玢 — 殇帝
  3. 刘晟 — 中宗
  4. 刘鋹 — 后主
7. 前蜀 (907–925) 偏安天府图享乐，荒淫无度速亡国。
  1. 王建 — 高祖
  2. 王衍 — 后主
8. 后蜀 (934–965) 闭关锁国醉太平，十四万人齐解甲。
  1. 孟知祥 — 高祖
  2. 孟昶 — 后主
9. 南平 (924–963) 弹丸之地凭无赖，四面称臣求生存。
  1. 高季兴 — 武信王
  2. 高从诲 — 文献王
  3. 高保融 — 贞懿王
  4. 高保勖 — （侍中）
  5. 高继冲 — （末代国君）
10. 北汉 (951–979) 依附契丹抗中原，十国遗脉终归宋。
  1. 刘崇 — 世祖
  2. 刘承钧 — 睿宗
  3. 刘继恩 — 少主
  4. 刘继元 — 英武帝
