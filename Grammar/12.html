一般将来时

一般将来时是英语时态中的一个重要组成部分，它用来描述或预测将要发生的动作或存在的状态。以下是一般将来时的几种常见结构及其用法，整理成条理清晰的形式：

1. 基本结构：shall/will + 动词原形

  • shall 通常用于第一人称（I或we），表示承诺、警告、命令或决心。
  • will 可用于所有人称，表示将来的简单动作或状态，或对未来的预测。
  • 缩略形式：'ll（例如：I will → I'll），won't（will not）和shan't（shall not，主要用于英式英语）。

2. be going to do sth. / be gonna do sth.（美语）

  • 表示打算或计划要做某事。
  • be going to 与will 通常可以互换，但在某些情况下，两者不能互换，具体取决于句子的具体含义。
  • 例如：I’ll miss you.（不能用be going to）；He'll lose.（不能用be going to）；I'll be sixteen years old next year.（不能用be going to）。

3. be + to do sth.

  • 表示计划安排在某时某地做某事，或用来征求意见。
  • 例如：I am to have a holiday.（我计划要休假。）

4. be about to do sth.

  • 表示即将做某事，强调动作的迫近性。
  • 例如：She is about to leave.（她即将离开。）

5. will be doing

  • 表示将来某个时间点正在进行的动作。
  • 例如：Tomorrow at this time, I will be flying to London.（明天这个时候，我将在飞往伦敦的途中。）

6. be doing（瞬间动词）

  • 使用现在进行时表示将来时态，这通常适用于某些特定的瞬间动词。
  • 适用的动词包括：go，come，arrive，leave，die，land，join。
  • 例如：He is coming next week.（他下周将来。）

7. be 一般现在时表示将要发生

  • 在某些情况下，一般现在时可以用来表示将来的动作，尤其是在条件句中。
  • 例如：If it rains, we will cancel the trip.（如果下雨，我们将取消旅行。）

通过以上整理，我们可以清晰地理解一般将来时的不同结构及其用法，这有助于我们在实际语言运用中更准确地表达未来的动作和状态。

be和set搭配副词的常见短语

be + 副词构成的表语

  1. be in
    • 表示在家或在某个地方。
    • 例句：She is in. 她在家。
  2. be out
    • 表示出去了，不在家或不在办公室。
    • 例句：He is out for lunch. 他出去吃午饭了。
  3. be away
    • 表示离开，不在自己的常驻地点。
    • 例句：They are away on vacation. 他们去度假了。
  4. be on
    • 表示上映、进行中或可用。
    • 例句：The movie is on at 7 p.m. tonight. 电影今晚7点上映。
  5. be back
    • 表示回来，返回。
    • 例句：I'll be back in an hour. 我一个小时后回来。
  6. be over
    • 表示结束，完成。
    • 例句：The meeting is over. 会议结束了。
  7. be up to sth.
    • 表示胜任某事，有能力做某事。
    • 例句：He is up to the challenge. 他有能力接受这个挑战。

set + 副词构成的短语动词

  1. set out
    • 表示出发，动身，开始做某事。
    • 例句：We set out at dawn to climb the mountain. 我们在黎明出发去爬山。
  2. set off
    • 表示出发，启程，通常指旅行或旅程的开始。
    • 例句：They set off for the trip early in the morning. 他们一大早就出发去旅行了。
  3. set up
    • 表示创立，建立，可以指公司、组织或设施。
    • 例句：She set up a new business. 她建立了一家新公司。
    • 也可以指创造记录。
    • 例句：He set up a new world record in swimming. 他在游泳中创造了新的世界记录。
