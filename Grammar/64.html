第三类条件句

用于设想过去纯粹想象的事情，并在主句中讲述想象的结果。这些假设情况是没有发生过的，或者永远不可能发生的。

基本结构：

if从句使用过去完成时（had + 过去分词），主句使用“would have / should have + 过去分词”。

例句：

  • If it had rained, we would have stayed at home.
  • 如果下雨了，我们就会呆在家里。
  • If I had worked harder at school, I'd have got a better job.
  • 如果我在学校更努力学习，我就会得到一份更好的工作。（实际情况是工作可能并不理想）

替代表达

在if从句中，可以使用“could have + 过去分词”来代替“had been able to + 动词原形”，使句子更加简洁。

例句：

  • I’d have read that book if I could have bought it.
  • 如果我能买到那本书，我就会读它。

第三类条件句是一种重要的语言工具，它帮助我们表达对过去的假设和可能的结果。通过使用这种结构，我们可以更准确地描述那些未曾发生或不可能发生的事情，以及这些事情对我们现实生活的影响。掌握第三类条件句的使用，可以丰富我们的语言表达，使我们的交流更加有效和精确。
