英语条件句的类型和用法
类型 结构 含义 例句
零条件句 if + 一般现在时, 一般现在时 表达事实或绝对真理 If water reaches 100 degrees, it boils.
第一条件句 if + 一般现在时, 一般将来时 表达将来可能发生的情况 If it rains, I will stay at home.
第二条件句 if + 一般过去时, would + 不定式 表达与现在事实相反或不太可能发生的情况 If I were rich, I would travel around the world.
第三条件句 if + 过去完成时, would have + 过去分词 表达与过去事实相反的情况 If I had studied harder, I would have passed the exam.
