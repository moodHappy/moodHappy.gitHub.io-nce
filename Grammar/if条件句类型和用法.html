英语条件句的类型和用法
类型 结构 含义 使用场景 例句 注意事项
零条件句 if + 一般现在时, 一般现在时 表达事实或绝对真理 用于描述普遍真理或科学事实 If water reaches 100 degrees Celsius, it boils. 这种条件句通常不涉及假设，而是陈述一个不变的事实。
第一条件句 if + 一般现在时, 一般将来时 表达将来可能发生的情况 用于描述基于特定条件下可能发生的将来事件 If it rains tomorrow, I will stay at home. 这种条件句通常用于讨论未来的假设情况。
第二条件句 if + 一般过去时, would + 不定式 表达与现在事实相反或不太可能发生的情况 用于描述与现在情况相反的假设 If I were rich, I would travel around the world. 这种条件句通常用于表达不切实际的愿望或假设。
第三条件句 if + 过去完成时, would have + 过去分词 表达与过去事实相反的情况 用于反思过去，假设过去的情况与实际不同 If I had studied harder, I would have passed the exam. 这种条件句用于表达对过去的遗憾或反思。
