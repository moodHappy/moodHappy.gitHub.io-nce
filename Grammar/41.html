英语情态动词

  1. 必须（must）

    用于表达义务或必要性，以及进行推测。
    • 对现在和将来的推测：使用must + 动词原形。例如：他现在一定已经在家了。
    • 对正在发生的事情的推测：使用must be doing。例如：她一定正在处理那个项目。
    • 对过去的推测：使用must have done。例如：他们一定早早就离开了。
    • 对过去正在发生的事情的推测：使用must have been doing。例如：我打电话时他一定在工作。

  2. 禁止（mustn't）

    用于表示绝对禁止，没有选择的余地。例如：你不能碰那根电线；它是带电的！

  3. 必须（have (got) to）

    与 "must" 类似，用于表达必须做某事的客观需要。其否定形式是 don't have to 或 haven't got to，表示没有义务或必要做某事。例如：如果你感觉不舒服，你不必参加会议。

  4. 需要（need）

    通常作为实义动词使用，表示需要。但在某些疑问句和否定句中，它可以作为情态动词使用，表达必要性。例如：你需要出去吗？或你不必为此担心。

  5. 回答疑问句

    当使用 "need" 作为情态助动词时，其一般疑问句的肯定回答可以用 must/had to，否定回答可以用 needn't。例如：我必须这么早就走吗？不，你不必。

  6. 表达不必要性

    needn't 或 don't have to 可以用来表达某事不是必要的，表示说话人给对方选择或允许对方不做某事。例如：你不必工作那么长时间。

  7. 一般疑问句的回答

    使用 "must" 的一般疑问句可以用 must/have to 或 needn't 来回答，但不使用 "mustn't"。例如：我今天必须完成这份报告吗？不，你不必。

  8. need 作为情态动词

    • 用于表达必要性，但不表示义务。其否定形式是 needn't，意为“不必”。例如：我需要出去吗？不，你不必。
    • 只能用于疑问句和否定句中，且在疑问句中通常期望得到否定的回答。
    • 在肯定句中不使用 need 作为情态动词，而是使用 must、have to、ought to 或 should。例如：今天上午我必须去看牙医。

  9. need 作为实义动词

    • 表示“需要”，可以用于肯定句、否定句和疑问句。其否定形式是 don't need，意为“不需要”。例如：我需要出去吗？是的，你需要。/不，你不需要。
    • 后面可以直接跟名词或不定式 to do。例如：他不需要跟着我。
    • 当 need 后面跟名词时，表示某物需要被做，使用主动语态表达被动含义。例如：你的鞋子需要洗。
    • need doing 或 need to be done 表达被动含义，其中 need doing 的主语通常是物，而 need to be done 的主语可以是人或物。例如：孩子需要被照看。
